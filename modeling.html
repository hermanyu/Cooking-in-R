Chapter 9 Modeling

Data usually comes with multiple variables and analyzing the covariation between these variables helps us understand the data. Model building is one method for analyzing this covariation. The model building process can be quite open-ended and freeform with an infinite number of design choices to be explored. In order to study all the different ways to build a model, it might be useful to formalize a basic framework. A formal framework allows us to study model design choices in a structured and systematic way.

Let us begin by formalizing what we want to study. The covariation between variables \(X_1,\ldots, X_k\) and \(Y\) is represented by the joint probability density function \(f:\mathbb{R}^{k+1}\to \mathbb{R}\). Heuristically, \(f(X_1,\ldots,X_n,Y)\) gives the probability of observing some given combination of \(X_1,\ldots,X_k\) and \(Y\) values. A model is a function \(\hat{f}:\mathbb{R}^{k+1}\to \mathbb{R}\) which approximates the joint density \(f\).

Often times, the scenario we want to model has \(Y\) being dependent on the value of \(X_1,\ldots,X_k\). In this case, what we really care about are the conditional probability distributions given by:

\[f(Y|X_1=x_1,\ldots,X_k=x_k) = f(x_1,\ldots,x_k, Y)\] The conditional distribution is the distribution of the random variable \(Y\) at the given values \(X_1=x_1,\ldots,X_k=x_k\). The idea is that \(Y\) has some inherent randomness; even if we knew that \(X_1=x_1,\ldots,X_k=x_k\), the value of \(Y\) can still fluctuate out of sheer random noise. For example, we can instinctively tell that a person’s weight \(Y\) should be dependent on that person’s height \(X\): the taller you grow, the more you weigh. Nonetheless, even two people with the same height can have different weights and this is what the conditional distribution reflects.

When \(Y\) is dependent on \(X\), we can model the joint density function \(f\) by modeling the individual conditional distributions \(f(Y|X_1,\ldots,X_n)\). How we build these models will depend on the use case. There are 2 main use cases for models:

  1. Inference: the model is used to explain a relationship between two variables. In this case, we are interested in the model building process itself and how components of the model correspond to real-world phenomena. The main focus in this situation is to evaluate the model building process and estimate how close our model \(\hat{f}\) actually is to the ground truth \(f\).

  2. Prediction: the model is used to generate outputs. In this case, we are interested in the outputs of the model and how those outputs look. The main focus for this situation is to get \(\hat{f}(X) \approx Y\) as close as possible.
